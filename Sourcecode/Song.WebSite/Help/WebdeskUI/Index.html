Webdesk UI 前端控件库

WebdeskUI，这个名字是我随便取的，可能会和别人重名吧。仓促写起，堆砌而成，懒得起名了。

仿桌面应用的后台管理模板，例如窗口控件，可拖放、可缩放、右键菜单、父子窗体互动、事件多播、双向绑定与监听。

原生Js架构（参考了Vue与JQuery编程思路），没有依赖第三方库。

原本没有打算自己写UI，毕竟开源且免费的UI库太多了，不过没有找到满足我需求的。免费的太弱鸡，功能强的要收费。 考虑到产品开发和应用，不希望将来在版权上受制于人，2020年春节疫情封控期间，十天左右完成。 不过由于精力有限，我只写了一些主要的控件。

其实我的需求并不复杂——我只是不希望我的B/S架构的管理系统看起来像个网页！
